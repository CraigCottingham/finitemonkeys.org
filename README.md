# finitemonkeys.org

Source code for the website at http://finitemonkeys.org.

## QUICK START

1. Clone the repository to a local workspace.
2. `gatsby develop`
3. Open http://localhost:8000/ to render the website.

## PUBLISH

1. `amplify publish`
